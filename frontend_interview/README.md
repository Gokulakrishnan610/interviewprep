# Interview Prep Frontend

A React-based frontend for the Interview Prep platform, designed specifically for students to practice interviews with AI agents.

## 🎯 Purpose

This application is designed for **students** to:
- Practice interviews with AI agents in a safe, judgment-free environment
- Improve communication skills through voice-based conversations
- Receive detailed feedback on their interview performance
- Prepare for real-world interviews with confidence

## Features

- **Student-Focused**: Designed exclusively for student self-learning and practice
- **Interview Sessions**: Pre-defined mock interview templates for structured practice
- **Practice Mode**: Free-form practice sessions with AI agents
- **Voice Interviews**: Real-time voice conversations with AI
- **User Authentication**: Secure login and registration with email verification
- **Progress Tracking**: Monitor your interview practice progress
- **Responsive Design**: Works on desktop and mobile devices

## User Roles

This application is designed for **students only**. All users are automatically assigned the STUDENT role and have access to:
- Interview practice sessions
- AI agent conversations
- Progress tracking
- Feedback and scoring

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
- `npm run eject` - Ejects from Create React App (not recommended)

## Technology Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **Axios** - HTTP client
- **Lucide React** - Icons
